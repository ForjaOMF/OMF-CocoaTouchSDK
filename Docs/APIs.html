List of available mobility services APIs

  • SMS Send
    • Sends SMS messages through an HTTP interface

  • Copiagenda
    • Gives you access to the Copiagenda service to retrieve your contact list through an HTTP interface

  • Localízame
    • Allows you to access Localizame service to locate our friends through an HTTP interface

  • Auto WapPush
    • Sends WapPush messages to your own terminal through an HTTP interface

  • MMS Send
    • Provides the capability of composing and sending MMS messages through an HTTP interface

  • SMS 2.0
    • Provides access to the SMS2.0 service that allows us to send instant messages to our contacts through an HTTP interface
