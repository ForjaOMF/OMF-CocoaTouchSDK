Copiagenda RetrieveContacts

Performs contact list retrieval

(NSDictionary*) RetrieveContacts: (NSString*) login password: (NSString*) pass;

Parameters
login
  String with user's telephone number

pass
  String with user's password

Return
Contact list stored for the user. The list is indexed by contact name. Each element in the list is a NSStringArray*. Name and telephone number of the contact will be stored in position 3 and 11 of the array respectively.
