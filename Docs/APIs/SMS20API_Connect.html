SMS20::Connect

Connects to SMS2.0 service

(NSDictionary*) Connect: (NSString*)login Nickname: (NSString*)nick;

Parameters
login
  String with user's telephone number

nick
  String with the nickname that we want to use (needed only first time)

Return
Contact list of the user indexed by UserId.

Each element in the list is a SMS20Contact*.

Below you can see a sample code to traverse the list: 
        NSDictionary *contactos;
        if ([contactos count] != 0) {
            id key;
            NSEnumerator * numerator = [contactos keyEnumerator];
            while (key = [numerator nextObject]) {
                SMS20Contact *Contact;
                Contact = [contactos objectForKey:key];
                //Do whatever you need with Contact
            }
        }
