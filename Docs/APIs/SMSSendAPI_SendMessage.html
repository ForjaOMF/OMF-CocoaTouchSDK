SMSSender SendMessage

Sends an SMS message

(NSString*) SendMessage: (NSString*) user password: (NSString*) pass destination: (NSString*) number message:(NSString*) text;

Parameters
user
  String with sender's telephone number

pass
  String with sender's password

number
  String with destination telephone number

text
  String with message text

Return
If the message is correctly sent, OK

else
  • 1 - Incorrect user
  • 4 - Service unavaliable
  • 5 - Incorrect password
  • 11 - Incorrect destination
  • 13 - Destinatation not specified
  • 14 - Destinatation not legible
  • 15 - Message not specified
