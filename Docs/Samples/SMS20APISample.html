SMS 2.0

A sample application that allows you to use SMS 2.0 service. It demonstrates:
  • SMS 2.0 API
To check this service you need 2 telephone numbers.
This sample application perform the following actions:
  • Connect to the service
  • Add contact
  • Send and recieve messages
  • Presence notification
  • Authorize contacts
  • Remove contact
  • Disconnect from service
